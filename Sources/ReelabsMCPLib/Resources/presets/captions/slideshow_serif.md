# Slideshow Serif

Georgia serif, regular weight, lower-third position. A step up in formality from `subtitle` — the serif gives it an editorial / documentary feel.

**Pairs with:** `narrated_slideshow`.

**Reach for something else when:**
- Content is modern / tech-flavored → `subtitle` (Helvetica).
- You want per-word karaoke → any preset with `highlightColor`.
