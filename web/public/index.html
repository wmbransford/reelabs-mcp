What it does

Word-level transcripts

Chirp speech-to-text with millisecond timestamps. Cuts snap to utterances, not guesses.

Kits, not prompts

Pick a recipe — talking-head, tutorial, interview, podcast clip — and the agent follows it verbatim.

Native AVFoundation render

Runs on your Mac. No uploads, no re-encoding round-trips, no third-party queue.

Karaoke captions

Word-synced highlights, custom fonts and colors. Burned in per-frame — no editor round-trip.

Silence removal

Auto-skip dead air. Drop the returned segments straight into a RenderSpec.

Your footage stays yours

Files never leave your machine. Only the transcript payload hits the cloud.

Install

Two commands. The CLI handles sign-in through your browser.

$ brew install reelabs/tap/reelabs-mcp && brew services start reelabs-mcp
$ reelabs-mcp sign-in

Then add to Claude

$ claude mcp add reelabs reelabs-mcp
