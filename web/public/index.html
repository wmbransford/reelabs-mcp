ReeLabs is an MCP server for Claude. Transcribe, cut, caption, and render video by describing the edit you want. Runs natively on your Mac — no uploads, no re-encoding, no subscription to a fourth cloud.

Install

$ brew install reelabs/tap/reelabs-mcp
$ reelabs-mcp sign-in

Then add to Claude

$ claude mcp add reelabs reelabs-mcp
